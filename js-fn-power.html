JS: The Power of Functions

Apr 19, 2016

In JS, function is the BOSS. It is the way to build classes (before ES6 it was the only way), defines scope, etc. ES6 changed this a bit, but still function is your most powerful tool in JS.

In JS you can pass function as argument to another function, or you can return it as result. Here is an example that uses both.

function forEach ( arr ) {
    function action ( fn ) {
        for ( var i in arr ) {
            fn ( arr [ i ] );
        }
        return action;
    }
    return action;
}

var x =0;
forEach ( [ 1 , 2 , 3 ] ) ( function ( i ) {
    x+=i;
});
// x = 6

This function accepts an array as argument, then returns a function that accepts a function as arguments, and calls it for every item in array (with value as argument).

Note that action function will return itself, means you can chain it and do as many loops as you want on the array with different functions.
